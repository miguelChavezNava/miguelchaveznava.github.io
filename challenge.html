Coding in Cardiology Challenge

Overview

This project takes in a medical dataset of 12,000 patient records in the first 48 hours of their hospital stay and uses machine learning techniques to predict patient outcomes. It uses a 2012 version of this challenge dataset from Physionet, where the program generates feature vectors for each patient, gathering different data like gender and age as well as mean and standard deviation for time values. These features were then fed into a logistic regression model and achieved an AUROC score of 0.85.

Challenge Details

Key Features & Takeaways

  • Extracted over 50 features from the medical dataset, dealing with missing values and normalizing the data
  • Used Python and Sk-Learn to implement the machine learning model and evaluate its performance
  • Tested a range of C values through k-fold clustering to find the best regularization parameter
  • Generated the confusion matrix for the model and evaluated its performance on metrics like accuracy, precision, and recall
